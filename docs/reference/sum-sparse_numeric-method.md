# Sum of a sparse_numeric vector

Sum of a sparse_numeric vector

## Usage

``` r
# S4 method for class 'sparse_numeric'
sum(x, ..., na.rm = FALSE)
```

## Arguments

- x:

  A `sparse_numeric` object.

- ...:

  Ignored.

- na.rm:

  Logical; if TRUE, remove NA values in the stored entries.

## Value

Numeric scalar giving the sum of all elements (zeros included).

# Arithmetic operators for sparse_numeric vectors

Elementwise addition, subtraction, and multiplication using the standard
`+`, `-`, and `*` operators.

## Usage

``` r
# S4 method for class 'sparse_numeric,sparse_numeric'
e1 + e2

# S4 method for class 'sparse_numeric,sparse_numeric'
e1 - e2

# S4 method for class 'sparse_numeric,sparse_numeric'
e1 * e2
```

## Arguments

- e1, e2:

  `sparse_numeric` vectors.
